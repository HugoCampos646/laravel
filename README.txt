# Aplicación de Gestión de Alumnos y Proyectos

Este proyecto es una aplicación web hecha con Laravel para gestionar alumnos, proyectos y usuarios.
Forma parte de una práctica de desarrollo web.

La aplicación permite iniciar sesión y, una vez dentro, administrar toda la información desde un panel principal.

---

## Funcionalidades

La aplicación incluye:

- Registro e inicio de sesión de usuarios
- Cierre de sesión
- Protección de páginas para que solo entren usuarios logueados

### Gestión de Usuarios
- Ver lista de usuarios
- Crear nuevos usuarios
- Editarlos
- Borrarlos

### Gestión de Proyectos
- Crear proyectos
- Ver todos los proyectos
- Editarlos
- Eliminarlos
- Ver los detalles de cada proyecto

### Gestión de Alumnos
- Crear alumnos
- Ver el listado
- Editarlos
- Eliminarlos
- Paginación de resultados

---

## Otras cosas que tiene la aplicación

- Interfaz hecha con Tailwind
- Mensajes de confirmación al borrar
- Mensajes de éxito al guardar datos
- Traducciones (cambio de idioma)
- Relación entre proyectos y alumnos
